# POSTME
POSTME : POSTprocessing MEtagenomics assembly with hybrid data.

To use the tool, use the following command, 

    python postme.py \
        -o workingDirectory/ \
        -a mummerPath/ \
        -c contigs.fasta \
        -r longreads.fasta
